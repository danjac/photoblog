**/dj-a11y**

Audits the codebase for accessibility violations (WCAG 2.1 AA).

Scans templates and Python files. Checks forms, icons, HTMX interactions,
Alpine.js components, and semantic HTML. Reports VIOLATION, WARNING, and ADVISORY
findings. Read `docs/accessibility.md` for conventions used in this project.

Example:
  /dj-a11y
