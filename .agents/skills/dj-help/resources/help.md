**/dj-help [command]**

Lists all available dj-* commands, or shows detailed help for a specific command.

Without arguments: prints all commands with their one-line description.
With a command name: prints the full help for that command.
The `dj-` prefix is optional.

Examples:
  /dj-help              # list all commands
  /dj-help a11y         # show help for /dj-a11y
  /dj-help dj-a11y      # same thing
